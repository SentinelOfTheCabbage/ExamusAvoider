1) Первичная настройка
- Создайте телеграм-бота (напишите @BotFather в telegram)
- Скопируйте token вашего бота из диалога с @BotFather после создания бота
- Откройте файл config.json любым удобным текстовым редактором (подойдёт и блокнот)
- Замените значение токена поля "token" на скопированный из диалога с @BotFather (кавычки обязательны).
	Должно получиться что-то вроде:
	"token": "1251352:ASdoker23_423afS4DA234-09134ADFd"

2) Как заменить[либо добавить впервые] человека, чтобы он получал скрины?
- Откройте файл config.json
- Убедитесь что напротив "is_working_mode" стоит значение false. В противном случае смело заменяйте true на false
- Попросите вашего друга написать вашему боту
- Запустите файл "Office Updater.exe" и следуйте инструкции, отображаемой в консоли.

3) Горячие клавиши
- ctrl+alt+a - Activation - активация/деактивация режима отправки скриншотов
- ctrl+alt+s - Silent Mode - отключение исходящих звуков, при активации/деактивации режима [п.1] и отправке скриншотов
- ctrl+alt+c - Capture - отправка текущего содержимого экрана своему другу в Telegram
- ctrl+alt+q - Quit - полное закрытие приложения. Сопровождается безусловным двойным звуком

4) Перечень обязательных файлов для успешной работы данной программы
- "Office Updater.exe" - основное приложение, отвечающее за создание скринов
- "config.json" - конфигурации приложения.
- "on.wav", "off.wav", "click.wav" - соответственно звуки при активации/деактивации/создании скриншота.
